# Customer Churn Analytics System

## Project Overview
This project analyzes customer churn data to identify patterns and factors affecting customer retention.

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn
- Power BI

## Features
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Churn Visualization
- Customer Retention Insights
- Interactive Power BI Dashboard

## Project Structure
- Dashboard/ → Power BI dashboard files
- Data/ → Dataset files
- Images/ → Project screenshots
- Notebooks/ → Jupyter notebooks

## Key Insights
- Customers with month-to-month contracts showed higher churn rates.
- Fiber optic internet users had higher churn compared to DSL users.
- Electronic check payment method customers were more likely to churn.
